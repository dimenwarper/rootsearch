# rootsearch

Tools for searching root problems in science and engineering.

## Overview

`rootsearch` provides a collection of methods and utilities for finding roots (zeros) of functions, commonly arising in scientific and engineering applications.

## Features

- Root-finding algorithms for scalar and vector-valued functions
- Support for common methods (e.g., bisection, Newton-Raphson, secant method)
- Utilities for diagnosing and visualizing root problems
